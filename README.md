# Task Manager App

A simple and intuitive Task Management web application built with Vue 3 and TypeScript.  
The app allows users to create, edit, and organize tasks using a Kanban-style board.

## Purpose of the App

The purpose of the Task Manager App is to provide an easy-to-use interface for managing tasks and visualizing workflow.

Main goals:
- Practice building a Vue 3 application using **multiple components**
- Apply **TypeScript** with strict typing 
- Implement **state management** using Pinia
- Handle **form submission and validation**
- Practice **responsive UI design**
- Work collaboratively using **Git and GitHub**

---

## Features

- Create, edit, and delete tasks
- Change task status (To Do / In Progress / Done)
- Edit tasks via modal dialog
- Priority labels (low / medium / high)
- State management with Pinia
- Responsive design for desktop and mobile
- Persistent task state

## Local Development Setup & Configuration

### Prerequisites
- Node.js (recommended version: **20.x**)

### Installation

1. Clone the repository:
   ```bash 
   git clone https://github.com/Seuder147/vie_task_manager_app.git

2. Navigate to the project folder:

cd vie_task_manager_app

3. Install dependencies:

npm install

4. Start the development server:

npm run dev

5. Open the app in your browser:

http://localhost:5173

**Deleting a Task**

Select Delete from the task menu
